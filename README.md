# create-template

#### 安装教程

1.  建议安装pnpm以运行本项目，可执行如下命令

```shell
   npm i -g pnpm
   pnpm i
```

2. 若已安装pnpm,直接运行
```shell
   pnpm i
```

#### 使用说明

1.  在对应子模块package.json中选择脚本运行命令
